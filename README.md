# Ruby Recipe Box 

By Lauryn Davis + Riley Starnes 

A Ruby/Sinatra app that practices implementing many-to-many relationships with Active Record. 

##### Technologies Used

* Ruby,
* ActiveRecord
* PostgreSQL,
* rspec
* capybara
* sinatra
* HTML 
* CSS 
* Bootstrap 

##### Setup

Clone repo 
Run Postgres
In terminal: 

* `bundle install`  
* `rake db:create`
* `rake db:migrate`  
* `rake db:test:prepare`
* `ruby app.rb`
